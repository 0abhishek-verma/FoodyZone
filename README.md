## Installation ğŸ“¥
1. Clone the repository:
```bash
    git clone https://github.com/0abhishek-verma/FoodyZone.git
```
2. Navigate to the project directory:
```bash
    cd FoodyZone
```
3. Install dependencies:
```bash
    npm install
```
4. Start the development server:
```bash
    npm start
```
